Batch AI Analysis - User Manual
Step 1: Select AI Tool

Choose from OpenAI, RTK LLM, or DeepSeek for analysis. Each tool has different capabilities.

Step 2: Enter Filter or JQL
  • Saved Filter ID: Enter your Jira filter ID (e.g., 18907). Get it from your saved filter URL.
  • JQL Query: Write a JQL query like: project = ABC AND status = Open
Step 3: Load Issues

Click "Load Issues" to fetch issues from Jira. The system will download issue details and attachments.

Step 4: Analyze
  • Click "Analyze" on individual issues, or
  • Click "Analyze All" to process all issues

The system will analyze each issue including:

  • Issue description
  • All attachments (.txt files)
  • Compressed files (.zip, .tar, .tgz) - extracted and analyzed
Step 5: Update to Jira

After analysis, click "Update All to Jira" to add the AI analysis as a comment on each issue.

Note: You need to add JIRA_PASSWORD in your .env file for the Jira comment update to work.
